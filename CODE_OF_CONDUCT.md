# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Be respectful. Harassment, discrimination, and abusive behavior are not tolerated. In addition,
because OurChat handles other people's private conversations: **do not** post real exported chat
data, real names, or personal information in issues, PRs, or discussions. Use the synthetic sample.

Report unacceptable behavior privately to the maintainers.
